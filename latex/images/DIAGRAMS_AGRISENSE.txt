--------------------------------------------------------------------------------
Cách xuất ảnh cho LaTeX (bên thứ 3 hoặc tự export)
--------------------------------------------------------------------------------
- Mở file .drawio tại https://app.diagrams.net/ (hoặc Draw.io Desktop).
- File → Export as → PNG (hoặc PDF). Nên bật border/transparent tùy nền trang.
- Đặt tên gợi ý (đã gắn trong main.tex):
    agrisense-kien-truc-tong-the.png
    agrisense-luudo-setup-loop.png
    agrisense-luudo-sensor-mqtt.png
- Trong main.tex: \includegraphics tại fig:kien-truc, fig:luudo, fig:luudo-2.
